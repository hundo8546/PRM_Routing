  │                                                                  │

  │                                                                  │
  │  Retrieve[T2] → Analyze[T2] → Check[T2] → Synthesize[T2]        │
  │   Good           Good          BAD ← T2 misses this failure      │
  │                                                                  │
  │  Problem: same tier regardless of what actually goes wrong       │

  ┌──────────────────────────────────────────────────────────────────┐
  │ BAAR                                                             │
  │ (offline classifier, no runtime signal)                          │
  │                                                                  │
  │  Retrieve[T1] → Analyze[T2] → Check[T3] → Synthesize[T3]        │
  │   Good           Good          BAD ✓        Good (over-escalated)│

  │  Classifier saw step index=2 in training → always escalates here │
  │  Problem: pattern-matches trajectory structure, not actual error  │

  ┌──────────────────────────────────────────────────────────────────┐
  │ TRIM                                                             │

  │                                                                  │
  │  Problem: math PRM cannot score heterogeneous agent steps.       │
  │           No middle tier — jumps between cheap and expensive.    │
  └──────────────────────────────────────────────────────────────────┘
